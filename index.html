طراحی یه وبسایت چقدر طول میکشه؟

این پرسش هم یه جواب کلی نداره، باز بستگی به پروژه داره. از یک روز برای نصب و راه اندازی وردپرس شروع میشه. اما اگه نیاز به کدنویسی HTML و CSS باشه، بیشتر طول میکشه.

امکان طراحی وبسایت به شکل دلخواه خودم هست؟ یا فقط باید از قالب آماده استفاده کنم؟

بله، البته که همینطوره. هر طرحی که مد نظر شماست قابلیت طراحیش وجود داره. فقط کافیه یه توضیح کلی ازش ارایه بدید، با بالاترین کیفیت واستون طراحی میشه.
